# 🧠 记忆力大冒险

儿童益智翻牌记忆力小游戏，适合 6 岁左右的小朋友进行日常的记忆力放松和轻度益智训练。

## ✨ 游戏特色

- 🎴 4×2 练习 / 4×3 简单 / 4×4 挑战三种难度模式
- 🔍 魔法放大镜道具（每局限用 1 次）
- ⭐ 每日 5 颗星星体力系统（防沉迷）
- 🗣️ 中文语音播报
- 🎉 通关五彩纸屑特效
- 📱 响应式设计，支持手机/平板

## 🎮 玩法说明

1. 点击卡片翻面，记住图案位置
2. 翻开两张相同图案的卡片即可消除
3. 消除所有卡片即为通关
4. 每天有 5 颗星星，每局消耗 1 颗

## 🚀 在线体验

直接打开 `index.html` 即可游玩，或访问 GitHub Pages 部署地址。

## 📦 技术栈

- 纯 HTML + CSS + JavaScript（零依赖）
- Tailwind CSS (CDN)
- Web Speech API 语音合成
- Canvas 五彩纸屑动画
- LocalStorage 本地存储
